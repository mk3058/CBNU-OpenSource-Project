## 💻 How to Run
```
- clone repo
    git clone https://github.com/mk3058/CBNU-OpenSource-Project
- install dependencies
    npm i
- run
    npm start
    
 * Create ".env" file containing the following, in the project root folder
  -> "REACT_APP_SERVICE_KEY=$$$$ YOUR SERVICE KEY $$$$"
```
